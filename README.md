## KASE Repo Market Indicators Pipeline
Automated Python pipeline for collecting key repo market indicators from the Kazakhstan Stock Exchange, including TONIA, TRION, and TWINA.

The project scrapes financial market data, processes it, and stores the cleaned results in a SQL database. A version of this tool was developed during my internship at Baiterek National Managing Holding and is used for daily data collection.

Tech stack: Python, pandas, Selenium, BeautifulSoup, SQLAlchemy, MySQL

Key features:
Automated data collection from KASE
Data cleaning and transformation
SQL database integration
Daily financial market indicator tracking
Housing Finance Forecasting

## A forecasting project for predicting housing finance indicators for the next 7 years.
The program extracts historical data from a database, applies linear regression models to individual indicators, and saves forecasted values back into the database with a dedicated "Forecast" status.

Tech stack: Python, pandas, scikit-learn, SQLAlchemy, MySQL

Key features:
Database-driven forecasting workflow
Linear regression baseline models
Automated insertion of predicted values
Practical use case in financial and housing-sector analysis
